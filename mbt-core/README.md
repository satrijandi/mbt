# mbt-core

MBT (Model Build Tool) is a declarative ML pipeline framework for tabular machine learning.

## Installation

```bash
pip install mbt-core
```

## Quick Start

```bash
# Initialize a new project
mbt init my-ml-project

# Compile a pipeline
mbt compile churn_training_v1

# Run a pipeline
mbt run --select churn_training_v1
```
